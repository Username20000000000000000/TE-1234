How it works

1. Öffne game1.html über den Button „Start Game“.

2. Mit den Pfeiltasten (← →) steuerst du dein Auto.

3. Oben siehst du das deutsche Wort. Fahr über das passende englische Wort.

4. Richtig = Score +1 · Falsch = du verlierst ein Leben.
